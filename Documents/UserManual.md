## How to Play
### Menu controls
- W - Navigate up  
- A - Navigate left  
- S - Navigate down  
- D - Navigate right  
- Spacebar - Select

### Character Controls
- W - Move character up  
- A - Move character left  
- S - Move character down  
- D - Move character right  
- Spacebar - Attack  
- Left Shift - Run
